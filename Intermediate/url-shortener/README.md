# 🔗 URL Shortener

A tool that shortens long URLs into shorter, shareable links.

## 📖 Features

- Generate short links
- Redirect to original URL
- Track usage
- Store links in DB

---

## 🛠️ Tech Stack

- Next.js
- React
- Tailwind CSS
- Prisma / MongoDB

---

## 🚀 Getting Started

```bash
cd Intermediate/url-shortener
npm install
npm run dev
```
